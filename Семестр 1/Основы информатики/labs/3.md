# Задание 1

Код `trace.scm`:
```scheme
(define-syntax trace-ex
  (syntax-rules ()
    ((trace-ex expr)
     (begin
       (write 'expr)
       (let ((res expr))
         (begin
           (display " => ")
           (write res)
           (newline)))))))     
```

Код `1.rkt`:
```sheme
(load "trace.scm")

(define (zip . xss)
  (if (or (null? xss)
          (null? (trace-ex (car xss))))
      '()
      (cons (map car xss)
            (apply zip (map cdr (trace-ex xss))))))

(zip '(1 2 3) '(one two three))
```
