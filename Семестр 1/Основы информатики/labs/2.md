  (if (null? xs)

### Задание 3
```scheme
(define (iterate f x n)
  (if (= n 0)
      '()
      (iterate-lst f (list x) n)))

(define (iterate-lst f x n)
  (if (= n 1)
      x
      (iterate-lst f (cons (car x) (map f x)) (- n 1))))
```

### Задание 5
```scheme
(define (include x xs)
  (and (not (null? xs))
       (or (equal? (car xs) x)
           (include x (cdr xs)))))

(define (any? pred? xs)
  (include #t (map pred? xs)))

(define (all? pred? xs)
  (not (include #f (map pred? xs))))
```
